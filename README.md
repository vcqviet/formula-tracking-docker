# Formula Tracking Test Assignment

## Technology
1. Docker
2. API: NodeJs
3. Database: MongoDb
4. APP: ReactJs

## To setup local environment, please follow step by step at below:

1. Install Docker
2. Pull code from git url
3. Init gitsubmodule and pull code api, app

```
$ git submodule init
$ git submodule update

```

4. Run command to pull docker-image and start server, app, database
`$ docker-compose up -d
